# 📚 ระบบค้นหาสื่อการเรียนรู้ (Textbook Search)

เว็บไซต์สำหรับการสืบค้นและคัดกรองหนังสือเรียนและสื่อการเรียนรู้ บัญชีที่ 1, 2 และ 3 ของกระทรวงศึกษาธิการ ที่ออกแบบมาให้รวดเร็ว ใช้งานง่าย ไม่ซับซ้อน และรองรับการทำงานบนมือถือได้อย่างเต็มรูปแบบ

🌐 **เข้าใช้งานระบบได้ที่นี่:** [https://project-sy789.github.io/textbook-search/](https://project-sy789.github.io/textbook-search/)

## 🌟 ฟีเจอร์เด่น
- **ค้นหารวดเร็ว (Real-time):** พิมพ์ปุ๊บเจอเลย ไม่ต้องรอโหลดหน้าเว็บใหม่
- **ตัวกรองอัจฉริยะ:** คัดกรองหนังสือตามบัญชีรายชื่อ, กลุ่มสาระการเรียนรู้, ระดับชั้น, และสำนักพิมพ์ ได้ทันที
- **ตะกร้าหนังสือจำลอง:** สามารถจัดตะกร้าหนังสือ กำหนดจำนวนเล่ม เพื่อดูราคารวมทั้งหมดที่ต้องใช้งบประมาณจัดซื้อ
- **นำออกเป็น Excel:** กดปุ่มเดียวเพื่อเซฟรายการในตะกร้าออกมาเป็นไฟล์ `.xls` พร้อมส่งต่อให้รานค้าหรือทำเอกสารจัดซื้อได้ทันที
- **ข้อมูลอัปเดตเสมอ:** ระบบจะทำการดึงข้อมูลใหม่จากฐานข้อมูลตั้งต้นของกระทรวงฯ ให้**อัตโนมัติทุกเช้าวันจันทร์** (เวลา 07:00 น.) ทำให้ได้สื่อการเรียนรู้ที่เป็นปัจจุบันอยู่เสมอ

## 📱 วิธีการเริ่มต้นใช้งาน
1. **ค้นหาหนังสือ:** พิมพ์ชื่อหนังสือ, ชื่อผู้แต่ง, หรือชื่อสำนักพิมพ์ ในช่องค้นหาด้านบนสุด
2. **ใช้ตัวกรอง:** เลือกหมวดหมู่ต่างๆ ที่แถบด้านบนเพื่อผลลัพธ์ที่ตรงจุดมากขึ้น (ระบบมีข้อความแจ้งเตือนสีเหลือง หากเลือกบัญชีที่ 2 หรือ 3)
3. **จัดตะกร้า:** กดปุ่ม **"เข้าตะกร้า"** ที่หนังสือที่ต้องการ ระบบจะจำสินค้าไว้ให้ที่ไอคอน 🛒 มุมขวาบน
4. **ดูยอดรวม/นำออก:** กดที่ไอคอน 🛒 เพื่อตรวจสอบยอดรวมเงินที่ต้องใช้ กดแก้ไขเพิ่ม/ลดจำนวนเล่ม หรือกดปุ่ม "Export เป็น Excel" เพื่อโหลดไฟล์ข้อมูลสรุปเก็บไว้ในเครื่องรันไปใช้งานต่อได้เลย
