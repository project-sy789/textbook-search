# ระบบสืบค้นหนังสือเรียนและสื่อการเรียนรู้ 📚

[**เข้าสู่เว็บไซต์พื่อใช้งาน**](https://project-sy789.github.io/textbook-search/)
  
ระบบค้นหาและเปรียบเทียบข้อมูลหนังสือเรียนและสื่อการเรียนรู้ บัญชี 1, 2 และ 3 อ้างอิงจากฐานข้อมูลสำนักวิชาการและมาตรฐานการศึกษา กระทรวงศึกษาธิการ เว็บไซต์ถูกออกแบบให้ใช้งานง่าย ค้นหาได้รวดเร็วแบบ Real-time และรองรับการดูผ่านโทรศัพท์มือถือ

## ฟีเจอร์หลัก (Features)
- 🔍 **ค้นหาแบบรวดเร็ว:** ค้นหาชื่อหนังสือ, ชื่อผู้เรียบเรียง หรือ สำนักพิมพ์ ได้ทันทีโดยไม่ต้องโหลดหน้าเว็บใหม่
- 🏷️ **ระบบตัวกรองอัจฉริยะ (Smart Filters):** กรองข้อมูลตาม ประเภท, กลุ่มสาระการเรียนรู้, ระดับชั้น และ สำนักพิมพ์
- 💡 **อัปเดตอัตโนมัติ (Auto Updates):** มีระบบ GitHub Actions ทำหน้าที่ดึงข้อมูล (Scraping) ข้อมูลรายการหนังสือล่าสุดจากเว็บไซต์หลักทุกๆ สัปดาห์ เพื่อให้ข้อมูลในเว็บนี้สดใหม่อยู่เสมอ
- 📱 **การออกแบบที่ทันสมัย:** อินเตอร์เฟซแบบ Glassmorphism สวยงาม พร้อมภาพประกอบหนังสือที่ดึงมาจากฐานข้อมูล

## การทำงานของระบบ
1. **Frontend:** พัฒนาด้วย `HTML`, `CSS` (Vanilla, Glassmorphism Design) และ `JavaScript` เพียวๆ เพื่อให้เว็บมีขนาดเล็กและโหลดได้เร็วที่สุด
2. **Backend/Database:** ไม่มี Backend หรือ Database เก็บข้อมูลแยกตายตัว แต่ใช้ไฟล์ `data.json` ก้อนเดียวเป็นฐานข้อมูล ซึ่งโหลดพร้อมกับเว็บ
3. **Automation:** สคริปต์ `scripts/scraper.py` เขียนด้วย Python 3 ฝังอยู่ใน GitHub Actions จะทำงานทุกวันจันทร์ 07:00AM หากพบหนังสือใหม่ ระบบจะแก้ไฟล์ `data.json` และนำขึ้นเว็บให้ทันที (Self-Updating Data)

## วิธีรันสคริปต์อัปเดตข้อมูลด้วยตัวเอง (Manual Check)
หากคุณนำโค้ดไปรันที่เครื่องตัวเอง หรือต้องการอัปเดตข้อมูลปัจจุบันทันที:

```bash
# รันเช็คสถานะอัปเดตข้อมูล
python3 scripts/scraper.py
```
*(รองรับ Python 3 การดึงข้อมูลใช้แค่ `urllib` และ `html.parser` ที่มีติดมากับระบบพื้นฐานอยู่แล้ว ไม่ต้องลง Library เพิ่ม)*

## ลิขสิทธิ์ข้อมูล
ข้อมูลหนังสือและภาพประกอบเป็นของทางกระทรวงศึกษาธิการและสำนักพิมพ์ตามระบุในเว็บต้นทาง ระบบนี้จัดทำขึ้นเพื่อเป็นเครื่องมืออ้างอิงและค้นหาที่สามารถเข้าถึงผ่านมือถือหรืออุปกรณ์พกพาได้รวดเร็วยิ่งขึ้นเท่านั้น
