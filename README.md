# 📚 ระบบค้นหาสื่อการเรียนรู้ (OBEC Textbook Search & Budget Planner)

เครื่องมืออัจฉริยะสำหรับคุณครูและบุคลากรทางการศึกษา เพื่อค้นหาหนังสือเรียน สื่อการเรียนรู้ (บัญชี 1, 2, 3) ตามนโยบายเรียนฟรี 15 ปี พร้อมระบบ **จำลองการจัดตะกร้าหนังสือ คำนวณงบประมาณโรงเรียน และตรวจสอบระเบียบ สพฐ. โดยอัตโนมัติ** 

🌐 **เข้าใช้งานระบบได้ที่นี่:** [https://project-sy789.github.io/textbook-search/](https://project-sy789.github.io/textbook-search/)

---

## ✨ ความสามารถเด่น (Key Features)

### 1. 🔍 ระบบค้นหาและคัดกรองแบบ Real-time
ค้นหาหนังสือทันใจเพียงปลายนิ้วพิมพ์! กรองข้อมูลได้ละเอียดทั้งตามระดับชั้น กลุ่มสาระการเรียนรู้ ประเภทบัญชี และสำนักพิมพ์ สะดวกรวดเร็ว ไม่ต้องรอหน้าเว็บโหลดใหม่ 

### 2. 🧮 เครื่องมือจัดเตรียมงบประมาณระดับโรงเรียน (School Budget Planner)
ไม่ต้องปวดหัวจดใส่กระดาษอีกต่อไป! ระบบมาพร้อมฟังก์ชัน "ตั้งค่างบประมาณโรงเรียน":
- **กรอกจำนวนนักเรียนแบบเหมาทุกชั้น:** ใส่ข้อมูลนักเรียนตั้งแต่ปฐมวัยไปจนถึง ม.6 ได้พร้อมกันในหน้าต่างเดียว
- **คำนวณยอดจัดสรรรวม:** โปรแกรมจะนำยอดนักเรียนไปคูณเรทมาตรฐาน สพฐ. ของแต่ละชั้นแบบอัตโนมัติ เพื่อสร้างเป็นเงินกองกลางของโรงเรียนสำหรับใช้คำนวณตะกร้าสินค้า

### 3. 🚨 ผู้ช่วยตรวจสอบระเบียบ (OBEC Strict Mode)
อุ่นใจ ไร้กังวลเรื่องผิดระเบียบจัดซื้อ! ระบบมีออโต้บอทตรวจข้อกำหนด สพฐ. ให้ทันทีเมื่อหยิบของใส่ตะกร้า:
- **บังคับวิชาพื้นฐาน (บัญชี 1) ตามระดับชั้น:** ป้องกันการสั่งซื้อสื่อเพิ่มเติมในชั้นนั้นๆ จนกว่าจะเลือก "บัญชี 1 ของชั้นเรียนนั้นถ้วน 8 กลุ่มสาระหลัก"
- **คุมกำเนิดแบบฝึกหัด:** แจ้งเตือนและระงับปุ่มกดทันทีหากมีการเลือกซื้อแบบฝึกหัดให้เด็กปฐมวัย, เด็กมัธยม, หรือวิชาของเด็กประถมที่นอกเหนือจาก (ไทย, คณิต, อังกฤษ) 
- **เตือนหลักสูตรเก่า:** ขึ้นป๊อปอัพแจ้งเตือนหากวิชา คณิต/วิทย์/ภูมิศาสตร์ เป็นเล่มที่ไม่ใช่ฉบับปรับปรุง พ.ศ. 2560 

### 4. 📊 แดชบอร์ดสรุปยอด & Export เป็น Excel
- **แถบแสดงสถานะงบ:** สรุปยอดเงินรวมที่ได้รับจัดสรร เงินที่ซื้อไปแล้ว และ "งบประมาณเหลือจ่าย" แจ้งเตือนสีแดงทันทีถ้ายอดซื้อทะลุงบเรียนฟรี
- **ประมวลผลเป็นเอกสาร:** เมื่อจัดตะกร้าเสร็จจนพอใจ สามารถกดปุ่มออกบิล Export เซฟเป็นไฟล์ `.xls` ปริ้นท์ส่งร้านค้า หรือทำประกอบใบเบิกงบได้ชิลๆ

## 💻 วิธีการใช้งาน (สำหรับคุณครู)

1. **ตั้งค่าฐานนักเรียน:** กดที่ไอคอน ⚙️ มุมขวาบน กรอกจำนวนเด็กลงไปในแต่ละชั้น ระบบจะอัปเดตยอดเงินให้ทันที (โปรดอิงยอดเด็กจากระบบ DMC รอบ 10 พ.ย. หรือ 10 มิ.ย.)
2. **เลือกหนังสือ:** กดเลือกตัวกรอง และคลิกปุ่มตระกร้า โปรแกรมจะตรวจสอบให้ทันทีว่าผิดระเบียบหรือยอดเงินเกินหรือไม่
3. **ตรวจสอบตะกร้า:** กดที่ไอคอนตะกร้า 🛒 ตรวจสอบว่ามี Alert สีแดง (ผิดระเบียบห้ามเบิก) หรือสีเหลือง (เตือนความจำ) โผล่ขึ้นมาหรือไม่ และกดเพิ่ม/ลดจำนวนเล่มได้เลย
4. **นำออกเป็นเอกสาร:** เมื่อยอดเงินถูกต้องแล้ว กด "Export เป็น Excel" ได้เลย! ระบบจะรวมยอดราคาสุทธิทุกรายการให้ในไฟล์
