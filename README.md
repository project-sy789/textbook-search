เครื่องมืออัจฉริยะสำหรับคุณครูและบุคลากรทางการศึกษา เพื่อค้นหาหนังสือเรียน สื่อการเรียนรู้ (บัญชี 1, 2, 3) ตามนโยบายเรียนฟรี 15 ปี พร้อมระบบ **จำลองการจัดตะกร้าหนังสือ คำนวณงบประมาณโรงเรียน และตรวจสอบระเบียบ สพฐ. โดยอัตโนมัติ**

🌐 **เข้าใช้งานระบบได้ที่นี่:** [https://project-sy789.github.io/textbook-search/](https://project-sy789.github.io/textbook-search/)

## ✨ ความสามารถเด่น (Key Features)

### 1. 🔍 ระบบค้นหาและคัดกรองแบบ Multi-Select
ค้นหาหนังสือทันใจ กรองข้อมูลได้แบบ **เลือกหลายรายการพร้อมกัน** ในแต่ละหมวด:
- **บัญชีรายชื่อ** — เลือก บัญชี 1 + บัญชี 2 พร้อมกัน
- **ประเภท** — หนังสือเรียน + แบบฝึกหัด พร้อมกัน
- **กลุ่มสาระ** — วิทยาศาสตร์ + คณิตศาสตร์ กรองมาพร้อมกันได้เลย
- **ระดับชั้น** — ป.1 + ป.2 พร้อมกัน
- **สำนักพิมพ์** — เลือกหลายสำนักพิมพ์พร้อมกัน

### 2. 🧮 เครื่องมือจัดเตรียมงบประมาณระดับโรงเรียน
กดปุ่ม ⚙️ เพื่อตั้งค่างบประมาณ:
- **กรอกจำนวนนักเรียนแต่ละชั้น** ระบบจะนำไปคูณเรทมาตรฐาน สพฐ. อัตโนมัติ
- **คำแนะนำ DMC พร้อมรายละเอียด** — กดปุ่ม [?] เพื่อดูวิธีคำนวณยอดนักเรียนตามภาคเรียน (รอบ 70% / 100%) และความแตกต่างระหว่างชั้นเคลื่อนกับชั้นเข้าใหม่

### 3. 🚨 ผู้ช่วยตรวจสอบระเบียบ OBEC Strict Mode (ครอบคลุมทุกระดับชั้น)

ระบบตรวจสอบแบบรายระดับชั้น ก่อนอนุญาตให้ซื้อบัญชี 2-3:

| ระดับชั้น | เงื่อนไขบัญชี 2-3 |
|----------|-----------------|
| **ปฐมวัย** | งบโรงเรียนไม่ติดลบ |
| **ประถม (ป.1-6)** | บัญชี 1 ครบ 8 กลุ่มสาระ + จำนวนเล่มครอบนักเรียนทุกคน + แบบฝึกหัด 3 วิชาบังคับ (ไทย/คณิต/อังกฤษ) |
| **มัธยม (ม.1-6)** | บัญชี 1 ครบ 8 กลุ่มสาระ + จำนวนเล่มครอบนักเรียนทุกคน |

> **หมายเหตุ:** เมื่อเงื่อนไขผ่านครบ ระบบจะแจ้งเตือน (สีเหลือง) ว่า _"ต้องผ่านความเห็นชอบของคณะกรรมการภาคี 4 ฝ่าย และกรรมการสถานศึกษาขั้นพื้นฐานก่อนดำเนินการ"_

**กฎอื่นๆ ที่ระบบตรวจให้อัตโนมัติ:**
- 🚫 ห้ามซื้อแบบฝึกหัดสำหรับปฐมวัยและมัธยม
- 🚫 ประถมอนุญาตแบบฝึกหัดเฉพาะ ไทย, คณิต, อังกฤษ เท่านั้น
- ⚠️ แจ้งเตือนหนังสือที่อาจไม่ใช่ฉบับปรับปรุง พ.ศ. 2560 (คณิต/วิทย์/ภูมิศาสตร์)
- ⚠️ แจ้งเตือนเมื่อเลือกบัญชี 2 หรือ 3 พร้อมคำอธิบายข้อระมัดระวังทางงบประมาณ
- 🔴 ปุ่ม "เข้าตะกร้า" จะเป็นสีแดงกดไม่ได้ทันทีสำหรับแบบฝึกหัดที่ผิดระเบียบ (ก่อนถึงตะกร้า)

### 4. 📦 ตะกร้าสินค้าพร้อมช่อง "หนังสือมีอยู่แล้ว/บริจาค"
แต่ละรายการในตะกร้ามี **2 ช่องแยกกัน:**
- **สั่งซื้อ** — จำนวนที่จะสั่งซื้อจริง (นับในงบประมาณ)
- **มีอยู่แล้ว** — หนังสือบริจาค/เวียน/ของเดิม (ไม่นับในงบ)
- **รวมทั้งหมด** = สั่งซื้อ + มีอยู่แล้ว → ระบบจะตรวจสอบว่ารวมแล้ว ≥ จำนวนนักเรียน

รองรับโรงเรียนที่มีระบบเวียนหนังสือหรือรับบริจาค โดยไม่ต้องสั่งซื้อครบ 100% ก็ผ่านระบบตรวจสอบได้

### 5. 📊 แดชบอร์ดสรุปยอด & Export Excel
- แถบแสดงงบจัดสรร / ใช้ไปแล้ว / เหลือจ่าย แบบ Real-time
- Export เป็นไฟล์ `.xls` พร้อมยอดรวมสุทธิ สำหรับทำใบเบิกงบ

## 💻 วิธีการใช้งาน

1. **ตั้งค่าฐานนักเรียน** — กด ⚙️ กรอกจำนวนนักเรียนแต่ละชั้น (อิงยอด DMC ณ วันที่ 10 พ.ย. หรือ 10 มิ.ย.)
2. **ค้นหาหนังสือ** — ใช้ตัวกรอง Multi-select เลือกได้หลายรายการพร้อมกัน
3. **เพิ่มเข้าตะกร้า** — กดปุ่ม "เข้าตะกร้า" (ปุ่มสีแดงกดไม่ได้ = ผิดระเบียบทันที)
4. **ปรับจำนวน** — ในตะกร้า ตั้ง "สั่งซื้อ" และ "มีอยู่แล้ว" ให้ถูกต้อง
5. **ตรวจสอบ Alert** — ดู Error (แดง) และ Warning (เหลือง) ในตะกร้า
6. **Export** — กด "Export เป็น Excel" เมื่อยอดถูกต้องครบถ้วน
